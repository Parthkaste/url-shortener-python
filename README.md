URL Shortener (Python CLI)

This is a simple command-line based URL Shortener built using Python.  
It takes long URLs and converts them into short, easy-to-share codes.  
Users can also retrieve the original URL using the generated short code.

Features
- Convert long URLs into short codes
- Retrieve original URLs using short codes
- Avoid duplicate URL entries
- Store data locally using a JSON file
- Easy-to-use command-line interface

 Tech Stack
- Python
- JSON
- File Handling

 How It Works
- The user enters a long URL
- The program generates a random 5-character short code
- The short code and original URL are saved in a JSON file
- The user can later retrieve the original URL using the short code
